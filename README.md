# KeenKeeper

**"Friends to keep close in your life"**

---

## 📖 Description

**KeenKeeper** is a dedicated relationship management tool designed to help users nurture and track their most important personal connections. It provides a centralized dashboard to visualize your inner circle and maintain a consistent log of interactions to ensure no friendship is left behind.

---

## 🛠️ Technologies

- **Next.js:** The core framework used for building the application and managing optimized image delivery.

- **Tailwind CSS:** Used to create the clean, modern card-based UI and responsive layouts.

- **React Icons:** Provides the social media icons and other UI elements for a polished look.

- **Mock API:** A simple mock API is used to simulate fetching friend data, which can be easily replaced with a real backend in the future.

---

## ✨ Key Features

**Friendship Dashboard:** A visual gallery under the "Your Friends" section that displays individual profile cards with quick-action status indicators
**Interaction Timeline:** A chronological "Timeline" that logs every touchpoint, including Meetups, Texts, Calls, and Video chats, to keep your communication history organized.
**Friendship Analytics:** A dedicated "Friendship Analytics" suite that tracks your social habits and identifies which connections might need more attention.
